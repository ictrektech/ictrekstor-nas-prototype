# PRD: 磁盘详情

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/storage/disks/detail/:deviceName` |
| 菜单标题 | 磁盘详情（隐藏菜单） |
| 页面权限 | 已登录用户 |
| 页面类型 | 详情页（子页面） |

## 需求概述

**总体功能**：磁盘详情页展示单块物理磁盘的详细信息，包括基本信息、SMART 健康状态、性能监控图表等。通过路由参数 `deviceName` 定位具体磁盘。

**目标用户**：需要了解单块磁盘健康状况和性能的技术用户或管理员。

**核心价值**：提供磁盘级别的深度信息，帮助用户及时发现硬盘故障风险，进行预防性维护。

---

## 功能规格

### 功能模块: 磁盘详情头部

#### 功能: 磁盘头部信息
- **功能描述**：页面顶部展示磁盘的核心标识信息和状态。
- **预期行为**：
  - 显示返回按钮（返回磁盘列表/设备管理页）
  - 显示磁盘名称（如 `/dev/sda`）
  - 显示磁盘型号、序列号
  - 显示状态标签（正常/警告/故障），颜色区分
- **约束条件**：
  - 状态基于 SMART 综合评估
- **可用场景**：快速确认磁盘身份和当前健康状态。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName` — 获取磁盘基本信息

---

### 功能模块: 设备信息卡片

#### 功能: 磁盘技术参数
- **功能描述**：以卡片或表格形式展示磁盘的详细技术参数。
- **预期行为**：
  - 显示设备名称、型号、序列号
  - 显示容量、接口类型（SATA/SAS/NVMe）
  - 显示固件版本、通电次数、通电时间
  - 显示当前温度
- **约束条件**：
  - 部分参数（如温度）实时变化，定时刷新
- **可用场景**：了解磁盘的硬件规格和使用历史。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/info` — 获取磁盘详细信息

---

### 功能模块: SMART 健康面板

#### 功能: SMART 综合评分
- **功能描述**：展示基于 SMART 数据的综合健康评分。
- **预期行为**：
  - 显示综合健康评分（百分比形式）
  - 评分低于阈值时显示警告色
  - 显示关键 SMART 属性摘要（如重映射扇区数、待映射扇区数）
- **约束条件**：
  - 评分算法由系统 SMART 模块计算
  - 不同厂商的 SMART 属性定义可能有差异
- **可用场景**：快速判断磁盘是否存在潜在故障风险。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/smart/summary` — 获取 SMART 综合评分

#### 功能: 关键 SMART 属性列表
- **功能描述**：展示对磁盘健康最关键的几项 SMART 属性。
- **预期行为**：
  - 列表展示关键属性：名称、当前值、阈值、状态
  - 异常属性以红色/橙色高亮
  - 点击查看完整 SMART 详情
- **约束条件**：
  - 仅展示关键属性，完整属性在弹窗中查看
- **可用场景**：了解具体哪些 SMART 指标出现异常。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/smart/attributes` — 获取关键 SMART 属性

---

### 功能模块: SMART 详情弹窗

#### 功能: 完整 SMART 属性表
- **功能描述**：以弹窗形式展示所有 SMART 属性的完整信息。
- **预期行为**：
  - 点击"查看详情"打开弹窗
  - 表格展示所有 SMART 属性
  - 每行包含：属性ID、名称、当前值、最差值、阈值、原始值、状态
  - 支持按状态筛选（正常/警告/异常）
  - 支持导出 SMART 报告
- **约束条件**：
  - 属性数量因磁盘厂商而异（通常约 20-30 项）
- **可用场景**：技术人员进行深度磁盘健康诊断。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/smart/all` — 获取完整 SMART 属性表

---

### 功能模块: 性能监控图表

#### 功能: 磁盘读写速率趋势
- **功能描述**：展示磁盘读写速率的历史趋势图。
- **预期行为**：
  - 显示读取速率和写入速率的曲线
  - 支持时间范围切换（最近1小时/24小时/7天）
  - 显示平均/峰值速率
  - 图表自动刷新（默认 5 秒）
- **约束条件**：
  - 数据来源于系统 I/O 监控
- **可用场景**：分析磁盘 I/O 性能，发现性能瓶颈。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/metrics/io` — 获取磁盘 I/O 时间序列数据

#### 功能: IOPS 趋势图
- **功能描述**：展示磁盘每秒 I/O 操作数（IOPS）的历史趋势。
- **预期行为**：
  - 显示读 IOPS 和写 IOPS 的曲线
  - 支持时间范围切换
  - 显示平均/峰值 IOPS
- **约束条件**：
  - 数据与读写速率趋势同步刷新
- **可用场景**：评估磁盘响应能力。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/metrics/iops` — 获取 IOPS 时间序列数据

#### 功能: 温度趋势图
- **功能描述**：展示磁盘温度的历史趋势。
- **预期行为**：
  - 显示温度变化曲线
  - 超过安全阈值时曲线区域变红
  - 支持时间范围切换
- **约束条件**：
  - 部分磁盘型号可能不支持温度监控
- **可用场景**：监控磁盘散热状况，预防过热损坏。
- **涉及接口**：
  - `GET /api/storage/disks/:deviceName/metrics/temperature` — 获取温度时间序列数据

## 预期交互

### 场景: 检查磁盘健康状态
1. 用户从设备管理页点击某硬盘"详情"按钮
2. 页面跳转至磁盘详情页，显示 `/dev/sda` 的信息
3. 头部显示状态为"正常"（绿色标签）
4. SMART 综合评分显示 98%，关键属性均正常
5. 用户点击"查看详情"，弹窗显示完整 SMART 属性表

### 场景: 分析磁盘性能
1. 用户向下滚动查看性能监控图表
2. 读写速率图显示当前读取 150MB/s，写入 80MB/s
3. 用户切换时间范围到"最近7天"，查看历史趋势
4. 温度图显示磁盘温度稳定在 42°C，未超过阈值

## 数据需求（Mock）

- **磁盘基本信息数据**：设备名称、型号、序列号、容量、接口类型、固件版本、通电次数、通电时间、温度
- **SMART 健康数据**：综合评分、关键属性列表（ID、名称、当前值、阈值、状态）
- **完整 SMART 属性表**：属性ID、名称、当前值、最差值、阈值、原始值、状态
- **性能监控时间序列数据**：读写速率、IOPS、温度的时间序列
