# PRD: 设备管理

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/device` |
| 菜单标题 | 设备管理 |
| 菜单图标 | `lucide:monitor` |
| 页面权限 | 已登录用户 |
| 页面类型 | 管理页（主页面） |

## 需求概述

**总体功能**：设备管理页是 NAS 系统的核心监控页面，展示设备硬件信息、资源使用情况、存储设备状态和网络配置。信息密度高且结构清晰。

**目标用户**：所有需要了解 NAS 设备硬件状态和性能的用户。

**核心价值**：作为设备状态的集中展示页面，让用户全面掌握硬件信息、资源使用、存储状态和网络配置。

---

## 功能规格

### 功能模块: 设备信息 Hero

#### 功能: 设备基本信息展示
- **功能描述**：页面顶部以 Hero 区域形式展示设备核心标识和运行信息。
- **预期行为**：
  - 显示设备名称、系统版本
  - 显示系统运行时长（开机至今）
  - 显示 CPU 核心数、内存大小
  - 显示硬盘数量
  - 显示设备序列号
  - 显示远程访问状态（启用/禁用）
- **约束条件**：
  - 信息从系统配置和硬件检测读取，不可在此页面编辑
  - 运行时长实时更新
- **可用场景**：快速确认设备身份和基本配置。
- **涉及接口**：
  - `GET /api/system/summary` — 获取设备摘要信息

---

### 功能模块: 硬件示意图

#### 功能: 设备前后视图
- **功能描述**：以 Canvas 绘制设备前后面板示意图，直观展示硬件布局。
- **预期行为**：
  - 前面板：盘位状态（空闲/已用/故障）、品牌标识
  - 后面板：风扇、网口、USB、HDMI、PCIe、电源接口
  - 盘位使用不同颜色标识状态（空闲=灰色/已用=蓝色/故障=红色）
  - 点击盘位高亮对应硬盘卡片
  - 鼠标悬停显示 Tooltip 详细信息
- **约束条件**：
  - 示意图基于设备型号预定义的布局模板
  - 不同设备型号的示意图可能不同
- **可用场景**：直观了解设备硬件布局和盘位使用情况。
- **涉及接口**：
  - `GET /api/device/diagram` — 获取设备硬件布局及盘位状态

#### 功能: 盘位与硬盘联动
- **功能描述**：硬件示意图中的盘位与下方硬盘卡片双向联动。
- **预期行为**：
  - 点击示意图中的盘位，下方硬盘列表自动滚动并高亮对应硬盘卡片
  - 点击硬盘卡片的"定位"按钮，示意图中对应盘位高亮闪烁
  - 硬盘状态变化时示意图对应盘位颜色同步更新
- **约束条件**：
  - 联动仅在当前页面有效
- **可用场景**：在物理设备上定位具体硬盘位置。
- **涉及接口**：无（纯前端交互）

---

### 功能模块: 存储设备

#### 功能: 硬盘卡片列表
- **功能描述**：以卡片形式展示设备中所有硬盘的信息。
- **预期行为**：
  - 每张卡片展示：盘位号、设备名、型号、容量
  - 显示温度、健康度（百分比）
  - 显示使用状态（已用/未使用）
  - 提供操作按钮：定位、休眠、闪烁、详情
- **约束条件**：
  - 硬盘信息从 SMART 和系统磁盘管理读取
  - 温度和健康度定时刷新
- **可用场景**：全面了解设备中各硬盘的状态。
- **涉及接口**：
  - `GET /api/storage/disks` — 获取硬盘列表

#### 功能: 硬盘操作
- **功能描述**：对单块硬盘执行管理操作。
- **预期行为**：
  - **定位**：在硬件示意图中高亮该硬盘所在盘位，方便物理定位
  - **休眠**：使硬盘进入休眠状态，降低功耗和噪音
  - **闪烁**：控制硬盘指示灯闪烁，用于在物理设备上快速定位
  - **详情**：跳转至 `/storage/disks/detail/:deviceName` 查看详细信息
- **约束条件**：
  - 定位/闪烁操作仅在设备支持时可用
  - 休眠操作可能影响存储池性能（RAID 重建期间不可用）
  - 已用于存储池的硬盘不可随意休眠
- **可用场景**：硬盘维护、物理定位、节能管理。
- **涉及接口**：
  - `POST /api/storage/disks/:deviceName/locate` — 定位硬盘
  - `POST /api/storage/disks/:deviceName/sleep` — 休眠硬盘
  - `POST /api/storage/disks/:deviceName/blink` — 闪烁硬盘指示灯

---

### 功能模块: 资源使用

#### 功能: CPU 使用率图表
- **功能描述**：展示 CPU 使用率的实时图表和历史趋势。
- **预期行为**：
  - 显示当前 CPU 使用率百分比
  - 展示最近一段时间的 CPU 使用率趋势曲线
  - 显示平均使用率
  - 图表自动刷新（默认 5 秒）
- **约束条件**：
  - 数据来源于系统性能监控
- **可用场景**：监控设备 CPU 负载。
- **涉及接口**：
  - `GET /api/system/metrics/cpu` — 获取 CPU 使用率时间序列数据

#### 功能: 内存使用率图表
- **功能描述**：展示内存使用率的实时图表和历史趋势。
- **预期行为**：
  - 显示当前内存使用率百分比和已用/总量
  - 展示最近一段时间的内存使用率趋势曲线
  - 显示平均使用率
  - 图表自动刷新（默认 5 秒）
- **约束条件**：
  - 数据定时刷新
- **可用场景**：监控设备内存使用情况。
- **涉及接口**：
  - `GET /api/system/metrics/memory` — 获取内存使用率时间序列数据

---

### 功能模块: 网络配置

#### 功能: 网卡卡片列表
- **功能描述**：以卡片形式展示设备中所有网络接口的信息。
- **预期行为**：
  - 每张卡片展示：网卡名称、MAC 地址
  - 显示 IP 地址、子网掩码
  - 显示连接状态（已连接/断开）
  - 显示实时速率（上行/下行）
  - 提供操作按钮：配置、详情
- **约束条件**：
  - 网络信息从系统网络管理模块读取
  - 实时速率定时刷新
- **可用场景**：查看和管理设备网络接口。
- **涉及接口**：
  - `GET /api/network/interfaces` — 获取网卡列表

#### 功能: 网卡操作
- **功能描述**：对单个网卡执行配置和查看操作。
- **预期行为**：
  - **配置**：打开网络配置弹窗，修改 IP 地址、子网掩码、网关、DNS 等
  - **详情**：跳转至 `/system/device/network/:name` 查看详细信息
- **约束条件**：
  - 修改网络配置可能导致网络断开，需二次确认
  - 部分配置变更需要重启网络服务生效
- **可用场景**：配置网络参数、排查网络问题。
- **涉及接口**：
  - `PUT /api/network/interfaces/:name` — 更新网卡配置

## 预期交互

### 场景: 查看设备整体状态
1. 用户进入"设备管理"页面
2. Hero 区域显示设备"NAS-Office"，运行 15 天 3 小时
3. 硬件示意图显示 6 个盘位中 5 个已用（蓝色），1 个空闲（灰色）
4. 下方硬盘卡片列表显示各盘位硬盘状态均正常

### 场景: 定位物理硬盘
1. 用户发现盘位 3 的硬盘温度偏高
2. 点击该硬盘卡片的"定位"按钮
3. 硬件示意图中盘位 3 高亮闪烁
4. 用户在物理设备上找到对应硬盘进行维护

### 场景: 配置网络
1. 用户发现某网口未连接
2. 点击该网卡卡片的"配置"按钮
3. 弹窗中检查 IP 配置，修改为正确的静态 IP
4. 保存后网络服务重启，网口恢复连接

## 数据需求（Mock）

- **设备基本信息数据**：名称、系统版本、运行时长、CPU核心数、内存大小、硬盘数量、序列号、远程访问状态
- **硬件盘位数据**：盘位号、状态、对应设备名
- **硬盘列表数据**：盘位号、设备名、型号、容量、温度、健康度、使用状态
- **资源使用时间序列数据**：CPU/内存使用率历史
- **网卡列表数据**：名称、MAC、IP、子网掩码、连接状态、实时速率
