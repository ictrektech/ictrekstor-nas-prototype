# PRD: 设备管理

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/device` |
| 菜单标题 | 设备管理 |
| 菜单图标 | `lucide:monitor` |

## 需求概述

设备管理页是 NAS 系统的核心监控页面，展示设备硬件信息、资源使用情况、存储设备状态和网络配置。信息密度高且结构清晰。

## 功能行为

### 1. 设备信息 Hero

展示：
- 设备名称、系统版本
- 运行时长
- CPU 核心数、内存大小
- 硬盘数量
- 设备序列号
- 远程访问状态

### 2. 硬件示意图

- Canvas 绘制的设备前后视图
- 前面板：盘位状态（空闲/已用/故障）、品牌标识
- 后面板：风扇、网口、USB、HDMI、PCIe、电源接口
- 点击盘位高亮对应硬盘
- 支持 Tooltip 显示详细信息

### 3. 存储设备

硬盘卡片列表，每张卡片展示：
- 盘位号、设备名
- 型号、容量
- 温度、健康度
- 使用状态（已用/未使用）
- 操作按钮：定位、休眠、闪烁、详情

### 4. 硬盘操作

| 操作 | 行为 |
|------|------|
| 定位 | 在硬件示意图中高亮该硬盘所在盘位 |
| 休眠 | 使硬盘进入休眠状态 |
| 闪烁 | 控制硬盘指示灯闪烁（用于物理定位） |
| 详情 | 跳转至 `/storage/disks/detail/:deviceName` |

### 5. 资源使用

- CPU 使用率实时图表（历史趋势）
- 内存使用率实时图表
- 显示平均使用率

### 6. 网络配置

网卡卡片列表，每张展示：
- 网卡名称、MAC 地址
- IP 地址、子网掩码
- 连接状态
- 实时速率
- 操作：配置、详情

### 7. 网卡操作

| 操作 | 行为 |
|------|------|
| 配置 | 打开网络配置弹窗修改 IP/DNS 等 |
| 详情 | 跳转至 `/system/device/network/:name` |

## 预期交互

- 页面加载时并行获取磁盘、存储池、网络、摘要数据
- 硬件示意图与硬盘卡片双向联动（点击盘位高亮卡片，点击卡片高亮盘位）

## 数据需求（Mock）

- 设备基本信息数据（名称、系统版本、运行时长、CPU、内存、硬盘数量、序列号、远程访问状态等）
- 硬件盘位数据（盘位号、状态、对应设备名）
- 硬盘列表数据（盘位号、设备名、型号、容量、温度、健康度、使用状态等）
- 资源使用时间序列数据（CPU/内存使用率历史）
- 网卡列表数据（名称、MAC、IP、子网掩码、连接状态、实时速率等）

## 关联页面

- `/storage/disks/detail/:deviceName` — 磁盘详情页
- `/system/device/network/:name` — 网卡详情页
