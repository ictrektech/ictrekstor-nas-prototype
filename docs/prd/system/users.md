# PRD: 用户管理

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/users` |
| 菜单标题 | 用户管理 |
| 菜单图标 | `lucide:users` |
| 页面权限 | 管理员 |
| 页面类型 | 管理页（主页面） |

## 需求概述

**总体功能**：用户管理页面用于管理 NAS 系统的本地用户账户，包括创建、编辑、删除用户，配置用户权限和配额等。

**目标用户**：NAS 系统管理员，负责用户账户管理和权限分配。

**核心价值**：提供统一的用户账户管理入口，确保系统访问安全，合理分配存储资源。

---

## 功能规格

### 功能模块: 用户列表

#### 功能: 用户表格展示
- **功能描述**：以表格形式展示所有本地用户账户。
- **预期行为**：
  - 表格列：用户名、角色、状态（启用/禁用）、创建时间、最后登录时间
  - 支持按用户名、角色、状态搜索和筛选
  - 支持按创建时间、最后登录时间排序
  - 支持分页展示
  - 点击用户行可展开详情或跳转详情页
- **约束条件**：
  - 仅管理员可查看所有用户列表
  - 系统内置账户（如 admin）可能不可删除
- **可用场景**：查看和管理系统中的所有用户账户。
- **涉及接口**：
  - 获取用户列表

---

### 功能模块: 用户账户操作

#### 功能: 创建用户
- **功能描述**：创建新的本地用户账户。
- **预期行为**：
  - 点击"创建用户"按钮，弹出表单弹窗
  - 表单字段：用户名（必填）、密码（必填）、确认密码（必填）、角色（下拉选择）、邮箱（可选）
  - 用户名不可与已有用户重复
  - 密码需满足复杂度要求（长度、字符类型）
  - 创建成功后用户列表自动刷新
- **约束条件**：
  - 仅管理员可创建用户
  - 用户名不可重复，不可使用系统保留名称
  - 密码需符合系统安全策略
- **可用场景**：为新加入团队的成员创建 NAS 账户。
- **涉及接口**：
  - 创建用户

#### 功能: 编辑用户
- **功能描述**：修改已有用户的信息和状态。
- **预期行为**：
  - 点击用户行的"编辑"按钮，弹出编辑弹窗
  - 可修改：邮箱、角色
  - 可重置密码（生成随机密码或手动设置）
  - 可启用/禁用账户
  - 保存后即时生效
- **约束条件**：
  - 仅管理员可编辑其他用户
  - 不可修改自己的角色为更低权限（防止失去管理员权限）
  - 禁用当前登录的管理员账户需二次确认
- **可用场景**：调整用户权限、重置遗忘密码、禁用离职员工账户。
- **涉及接口**：
  - 更新用户信息
  - 重置用户密码

#### 功能: 删除用户
- **功能描述**：删除用户账户。
- **预期行为**：
  - 点击用户行的"删除"按钮
  - 弹出二次确认弹窗，提示可选择是否保留用户文件
  - 确认后删除用户账户
  - 如选择保留文件，文件归属转移至管理员或保留在原路径
  - 刷新用户列表
- **约束条件**：
  - 仅管理员可删除用户
  - 系统内置账户不可删除
  - 不可删除当前登录的管理员账户
- **可用场景**：清理离职员工的账户。
- **涉及接口**：
  - 删除用户

---

### 功能模块: 用户详情

#### 功能: 用户详细信息
- **功能描述**：展示单个用户的详细信息和资源使用情况。
- **预期行为**：
  - 点击用户行展开或跳转详情页
  - 显示用户基本信息：用户名、角色、邮箱、状态、创建时间
  - 显示存储配额使用情况（已用/配额上限）
  - 显示共享文件夹权限列表
  - 显示最近登录历史（时间、IP）
- **约束条件**：
  - 普通用户仅可查看自己的详情
  - 管理员可查看所有用户详情
- **可用场景**：审计用户资源使用，排查访问问题。
- **涉及接口**：
  - 获取用户详情
  - 获取用户存储配额
  - 获取用户登录历史

---

### 功能模块: 用户组管理

#### 功能: 用户组列表
- **功能描述**：展示和管理用户组。
- **预期行为**：
  - 以表格展示用户组：组名、成员数、权限描述
  - 支持创建/编辑/删除用户组
  - 支持向组内添加/移除用户
- **约束条件**：
  - 仅管理员可管理用户组
  - 系统内置用户组（如 administrators、users）不可删除
- **可用场景**：通过用户组批量管理用户权限。
- **涉及接口**：
  - 获取用户组列表
  - 创建用户组
  - 更新用户组
  - 删除用户组

---

### 功能模块: 配额设置

#### 功能: 用户/组配额配置
- **功能描述**：为用户或用户组设置存储空间配额。
- **预期行为**：
  - 在用户详情或用户组详情中配置配额
  - 设置配额上限（GB/TB）
  - 显示当前已用容量和配额使用率
  - 达到配额阈值时发出警告
- **约束条件**：
  - 仅管理员可配置配额
  - 配额上限不能超过存储池可用容量
  - 已用容量超过新配额时，新配额不能低于已用容量
- **可用场景**：限制用户存储使用，合理分配存储资源。
- **涉及接口**：
  - 设置用户配额
  - 设置用户组配额

---

### 功能模块: 批量操作

#### 功能: 批量用户操作
- **功能描述**：对多个用户执行批量操作。
- **预期行为**：
  - 表格支持多选（Ctrl/Shift + 点击或复选框）
  - 批量操作按钮：启用、禁用、删除
  - 批量删除时弹出确认弹窗，列出受影响用户数量
- **约束条件**：
  - 批量操作中包含系统内置账户时，该账户的操作会被跳过
  - 批量删除不可删除当前登录账户
- **可用场景**：批量管理多个用户账户。
- **涉及接口**：
  - 批量更新用户状态
  - 批量删除用户

## 预期交互

### 场景: 创建新用户
1. 管理员点击"创建用户"按钮
2. 弹窗中输入用户名"zhangsan"、密码、确认密码
3. 选择角色"普通用户"，输入邮箱
4. 确认创建，列表自动刷新显示新用户

### 场景: 调整用户权限
1. 管理员在用户列表中找到"lisi"
2. 点击"编辑"，将角色从"普通用户"改为"管理员"
3. 保存后该用户立即获得管理员权限

### 场景: 配置存储配额
1. 管理员点击某用户行展开详情
2. 显示该用户已用 500GB，当前配额 1TB
3. 管理员将配额调整为 2TB
4. 用户侧的存储空间提示同步更新

### 场景: 批量禁用离职员工
1. 管理员选中 3 个离职员工的账户
2. 点击"批量禁用"
3. 弹窗确认后，3 个账户状态变为"禁用"
4. 被禁用的用户无法再登录系统

- **用户组列表数据**：组ID、组名、成员数、权限描述
- **用户详情数据**：基本信息、存储配额（已用/上限）、共享文件夹权限列表、登录历史（时间、IP）
