# PRD: 用户管理

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/users` |
| 页面组件 | `apps/web-antd/src/views/system/users/index.vue` |
| 菜单标题 | 用户管理 |
| 菜单图标 | `lucide:users` |
| 当前状态 | 🚧 施工中（占位页） |

## 需求概述

用户管理页面用于管理 NAS 系统的本地用户账户，包括创建、编辑、删除用户，配置用户权限和配额等。

## 功能行为

### 当前实现

页面目前为**施工占位页**，显示：
- 大号施工图标（`lucide:hard-hat`）
- "施工中" 标题
- "此功能正在开发中，敬请期待" 描述文本

### 预期功能（待实现）

1. **用户列表**：展示所有本地用户（用户名、角色、状态、创建时间、最后登录）
2. **创建用户**：弹窗表单，包含用户名、密码、确认密码、角色、邮箱等
3. **编辑用户**：修改用户信息、重置密码、启用/禁用账户
4. **删除用户**：确认后删除，可选择是否保留用户文件
5. **用户详情**：查看用户的存储配额、共享文件夹权限、登录历史
6. **用户组管理**：创建/编辑用户组，批量分配权限
7. **配额设置**：为用户或用户组设置存储空间配额

## 预期交互

- 表格支持排序、筛选、分页
- 点击用户行可展开详情或跳转详情页
- 批量操作支持多选后批量启用/禁用/删除

## 相关接口

| 接口 | 说明 |
|------|------|
| `GET /api/system/users` | 获取用户列表 |
| `POST /api/system/users` | 创建用户 |
| `PUT /api/system/users/:id` | 更新用户信息 |
| `DELETE /api/system/users/:id` | 删除用户 |
| `GET /api/system/groups` | 获取用户组列表 |
