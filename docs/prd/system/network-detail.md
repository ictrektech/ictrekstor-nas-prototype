# PRD: 网卡详情

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/device/network/:name` |
| 菜单标题 | 网卡详情（隐藏菜单） |

## 需求概述

网卡详情页展示单个网络接口的详细信息，包括基础配置、实时流量图表、历史流量统计等。通过路由参数 `name` 定位具体网卡。

## 功能行为

### 1. 基础信息

展示：
- 接口名称
- MAC 地址
- IP 地址 / 子网掩码 / 网关
- DNS 服务器
- 连接状态（已连接/断开）
- 连接速率（如 1000 Mbps）
- 双工模式

### 2. 流量监控图表

- 上行/下行速率实时曲线
- 支持时间范围切换
- 自动刷新（每 5 秒）

### 3. 数据统计

- 总接收字节数 / 总发送字节数
- 接收包数 / 发送包数
- 错误包数 / 丢弃包数
- 格式化显示（KB/MB/GB/TB/PB）

## 预期交互

- 点击"返回"按钮回到 `/system/device`
- 图表自动刷新展示实时流量
- 状态变化时标签颜色同步更新（已连接=绿色/断开=红色）

## 数据需求（Mock）

- 网卡基础信息数据（接口名称、MAC、IP、子网掩码、网关、DNS、连接状态、速率、双工模式）
- 实时流量时间序列数据（上行/下行速率）
- 累计统计数据（收发字节数、包数、错误包、丢弃包）
