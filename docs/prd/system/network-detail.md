# PRD: 网卡详情

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/system/device/network/:name` |
| 页面组件 | `apps/web-antd/src/views/system/device/network-detail.vue` |
| 菜单标题 | 网卡详情（隐藏菜单） |
| 当前状态 | ✅ 已实现 |

## 需求概述

网卡详情页展示单个网络接口的详细信息，包括基础配置、实时流量图表、历史流量统计等。通过路由参数 `name` 定位具体网卡。

## 功能行为

### 1. 页面布局

- **顶部**：返回按钮 + 网卡名称 + 状态标签
- **基础信息**：`Descriptions` 展示网卡技术参数
- **流量图表**：`vue-echarts` 绘制的实时带宽趋势图
- **统计信息**：收发包数量、错误包、丢包等

### 2. 基础信息

展示：
- 接口名称
- MAC 地址
- IP 地址 / 子网掩码 / 网关
- DNS 服务器
- 连接状态（已连接/断开）
- 连接速率（如 1000 Mbps）
- 双工模式

### 3. 流量监控图表

- 上行/下行速率实时曲线
- 使用 `LineChart` 绘制
- 支持时间范围切换
- 自动刷新（每 5 秒）

### 4. 数据统计

- 总接收字节数 / 总发送字节数
- 接收包数 / 发送包数
- 错误包数 / 丢弃包数
- 格式化显示（KB/MB/GB/TB/PB）

## 预期交互

- 点击"返回"按钮回到 `/system/device`
- 图表自动刷新展示实时流量
- 状态变化时标签颜色同步更新（已连接=绿色/断开=红色）

## 相关接口

| 接口 | 说明 |
|------|------|
| `GET /api/system/networks/:name` | 获取网卡详情 |
