## 首页/仪表盘

### 获取设备摘要信息

- **功能描述**: 获取设备的基本信息，用于首页顶部展示。
- **预期行为**: 返回设备名称、系统版本、运行时长、序列号等核心信息。
- **输入参数**: 无。
- **输出数据**:
  - `deviceName`: 设备名称（字符串）
  - `systemVersion`: 系统版本号（字符串）
  - `uptime`: 运行时长（字符串，如 "15 天 3 小时"）
  - `serialNumber`: 设备序列号（字符串）

### 获取 CPU 使用率时间序列数据

- **功能描述**: 获取 CPU 使用率的实时及历史趋势数据，用于绘制 CPU 使用率图表。
- **预期行为**: 返回一段时间内 CPU 使用率的采样数据。
- **输入参数**:
  - `timeRange`: 时间范围（枚举：1h, 6h, 24h, 7d）
- **输出数据**:
  - `timestamps`: 时间戳数组（字符串数组，ISO 8601 格式）
  - `usage`: CPU 使用率数组（数值数组，0-100）

### 获取内存使用率时间序列数据

- **功能描述**: 获取内存使用率的实时及历史趋势数据，用于绘制内存使用率图表。
- **预期行为**: 返回一段时间内内存使用率的采样数据。
- **输入参数**:
  - `timeRange`: 时间范围（枚举：1h, 6h, 24h, 7d）
- **输出数据**:
  - `timestamps`: 时间戳数组（字符串数组，ISO 8601 格式）
  - `usage`: 内存使用率数组（数值数组，0-100）

### 获取磁盘使用率数据

- **功能描述**: 获取各存储空间的使用率、总容量、已用容量，用于首页存储总览。
- **预期行为**: 返回每个存储空间/卷的使用情况概览。
- **输入参数**: 无。
- **输出数据**:
  - `volumes`: 存储空间列表，每项包含：
    - `name`: 存储空间名称（字符串）
    - `total`: 总容量（字符串，如 "2 TB"）
    - `used`: 已用容量（字符串）
    - `usagePercent`: 使用率百分比（数值，0-100）

### 获取存储池健康状态

- **功能描述**: 获取存储池的整体健康状态统计。
- **预期行为**: 返回存储池总数及正常、警告、故障数量。
- **输入参数**: 无。
- **输出数据**:
  - `total`: 存储池总数（数值）
  - `healthy`: 正常数量（数值）
  - `warning`: 警告数量（数值）
  - `error`: 故障数量（数值）

### 获取总容量数据

- **功能描述**: 获取设备整体的存储容量概览。
- **预期行为**: 返回设备总容量、已用容量、可用容量。
- **输入参数**: 无。
- **输出数据**:
  - `total`: 总容量（字符串，如 "8 TB"）
  - `used`: 已用容量（字符串）
  - `free`: 可用容量（字符串）
  - `usagePercent`: 使用率百分比（数值，0-100）

### 获取网口状态

- **功能描述**: 获取各网口的连接状态及速率信息。
- **预期行为**: 返回每个网口的连接状态、上行/下行速率。
- **输入参数**: 无。
- **输出数据**:
  - `interfaces`: 网口列表，每项包含：
    - `name`: 网口名称（字符串，如 "LAN1"）
    - `connected`: 是否已连接（布尔值）
    - `upRate`: 上行速率（字符串，如 "100 MB/s"）
    - `downRate`: 下行速率（字符串）

### 获取最近通知列表

- **功能描述**: 获取系统最近的通知/告警信息列表。
- **预期行为**: 返回最近的系统通知列表，按时间倒序排列。
- **输入参数**:
  - `limit`: 返回条数限制（数值，默认 10）
- **输出数据**:
  - `notifications`: 通知列表，每项包含：
    - `id`: 通知 ID（字符串）
    - `type`: 通知类型（枚举：info, warning, error）
    - `title`: 通知标题（字符串）
    - `message`: 通知内容（字符串）
    - `time`: 通知时间（字符串）
    - `read`: 是否已读（布尔值）

### 标记通知已读

- **功能描述**: 将指定通知标记为已读状态。
- **预期行为**: 更新指定通知的已读状态，成功后返回确认。
- **输入参数**:
  - `id`: 通知 ID（字符串）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 清除通知

- **功能描述**: 清除指定或全部通知。
- **预期行为**: 删除指定 ID 的通知，或当未指定 ID 时清除全部通知。
- **输入参数**:
  - `id`: 通知 ID（字符串，可选，不传则清除全部）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

## 文件管理

### 获取用户存储空间列表

- **功能描述**: 获取当前用户有权限访问的存储空间列表，作为文件系统根路径的入口。
- **预期行为**: 返回用户可访问的存储空间（卷）列表，用于定位文件系统的顶层路径。
- **输入参数**: 无。
- **输出数据**:
  - `volumes`: 存储空间列表，每项包含：
    - `id`: 存储空间 ID（字符串）
    - `name`: 存储空间名称（字符串）
    - `path`: 文件系统路径（字符串，如 "/volume1"）
    - `total`: 总容量（字符串）
    - `used`: 已用容量（字符串）

### 获取文件列表

- **功能描述**: 获取指定目录下的文件和文件夹列表，支持平铺列表和树形结构两种返回模式，适用于所有文件管理场景（我的文件、公共文件、全局浏览等）。
- **预期行为**: 根据当前选中的目录路径返回该目录下的文件和文件夹列表。当启用树形模式时，递归返回文件夹层级结构，供左侧目录树面板使用。
- **输入参数**:
  - `path`: 当前目录路径（字符串，可选，不传则返回所有可访问存储空间的根节点）
  - `volumeId`: 所属存储空间 ID（字符串，可选）
  - `deviceId`: 外接设备 ID（字符串，可选，指定时返回该外接设备路径下的文件列表）
  - `mode`: 返回模式（枚举：list, tree，默认 list）
  - `page`: 页码（数值，可选，仅 list 模式有效）
  - `pageSize`: 每页条数（数值，可选，仅 list 模式有效）
- **输出数据**:
  - **list 模式**（默认）：
    - `items`: 文件/文件夹列表，每项包含：
      - `id`: 项目 ID（字符串）
      - `name`: 名称（字符串）
      - `type`: 类型（枚举：file, folder）
      - `size`: 文件大小（字符串，文件夹为空）
      - `modifiedTime`: 修改时间（字符串）
    - `total`: 总条数（数值，可选）
  - **tree 模式**：
    - `tree`: 树节点数组，每项包含：
      - `id`: 节点 ID（字符串）
      - `name`: 节点名称（字符串）
      - `type`: 节点类型（枚举：volume, folder）
      - `path`: 节点对应的路径（字符串）
      - `children`: 子节点数组（可选）

### 创建文件夹

- **功能描述**: 在指定目录下创建新文件夹，适用于所有文件管理场景。
- **预期行为**: 在指定路径下创建文件夹，成功后返回新文件夹信息。
- **输入参数**:
  - `path`: 父目录路径（字符串）
  - `name`: 新文件夹名称（字符串）
- **输出数据**:
  - `id`: 新文件夹 ID（字符串）
  - `name`: 文件夹名称（字符串）
  - `createdTime`: 创建时间（字符串）

### 重命名文件/文件夹

- **功能描述**: 修改文件或文件夹名称，适用于所有文件管理场景。
- **预期行为**: 更新指定文件/文件夹的名称，成功后返回确认。
- **输入参数**:
  - `id`: 文件/文件夹 ID（字符串）
  - `newName`: 新名称（字符串）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 搜索文件

- **功能描述**: 按文件名在全局范围搜索文件。
- **预期行为**: 根据关键词在所有存储空间中搜索匹配的文件/文件夹。
- **输入参数**:
  - `keyword`: 搜索关键词（字符串）
  - `scope`: 搜索范围（枚举：all, currentVolume，可选）
- **输出数据**:
  - `results`: 搜索结果列表，每项包含：
    - `id`: 项目 ID（字符串）
    - `name`: 名称（字符串）
    - `path`: 所在路径（字符串）
    - `type`: 类型（枚举：file, folder）

### 下载文件

- **功能描述**: 将指定文件下载到本地。
- **预期行为**: 返回文件的下载链接或直接触发下载流。
- **输入参数**:
  - `id`: 文件 ID（字符串）
- **输出数据**:
  - `downloadUrl`: 下载链接（字符串）

### 批量下载

- **功能描述**: 支持批量下载多个文件，通常打包为压缩包。
- **预期行为**: 接收多个文件 ID，返回打包后的下载链接。
- **输入参数**:
  - `ids`: 文件 ID 数组（字符串数组）
- **输出数据**:
  - `downloadUrl`: 打包下载链接（字符串）

### 获取外接设备列表

- **功能描述**: 获取当前连接到 NAS 的所有外接设备列表。
- **预期行为**: 返回外接存储设备（USB、SD 等）的列表。
- **输入参数**: 无。
- **输出数据**:
  - `devices`: 外接设备列表，每项包含：
    - `id`: 设备 ID（字符串）
    - `name`: 设备名称（字符串）
    - `type`: 设备类型（枚举：usb, sd）
    - `capacity`: 总容量（字符串）
    - `used`: 已用容量（字符串）
    - `status`: 状态（枚举：mounted, unmounted）

### 获取外接设备详情

- **功能描述**: 获取当前外接设备的基本信息，用于外接设备文件管理页面顶部展示。
- **预期行为**: 返回指定外接设备的详细信息。
- **输入参数**:
  - `deviceId`: 设备 ID（字符串）
- **输出数据**:
  - `id`: 设备 ID（字符串）
  - `name`: 设备名称（字符串）
  - `type`: 设备类型（字符串）
  - `capacity`: 总容量（字符串）
  - `used`: 已用容量（字符串）
  - `fileSystem`: 文件系统类型（字符串）

### 获取设备目录树（外接设备）

- **功能描述**: 展示外接设备的文件目录树结构。
- **预期行为**: 返回指定外接设备的目录层级结构。
- **输入参数**:
  - `deviceId`: 设备 ID（字符串）
- **输出数据**:
  - `tree`: 树节点数组，结构同「获取文件列表」的 tree 模式

### 获取本地目录树

- **功能描述**: 展示 NAS 本地存储的文件目录树，用于外接设备管理页面的本地侧边。
- **预期行为**: 返回本地存储空间的目录树结构。
- **输入参数**: 无。
- **输出数据**:
  - `tree`: 树节点数组，结构同「获取文件列表」的 tree 模式

### 安全弹出设备

- **功能描述**: 安全卸载已连接的外接设备，防止数据损坏。
- **预期行为**: 执行卸载操作，卸载成功后设备状态变为未连接。
- **输入参数**:
  - `deviceId`: 外接设备 ID（字符串）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 上传文件

- **功能描述**: 上传文件到指定目录。
- **预期行为**: 接收文件流并保存到指定路径，支持分片上传和进度反馈。
- **输入参数**:
  - `path`: 目标路径（字符串）
  - `file`: 文件数据（二进制流）
- **输出数据**:
  - `success`: 是否上传成功（布尔值）
  - `fileId`: 新文件 ID（字符串）

### 移动/复制文件

- **功能描述**: 将文件或文件夹移动或复制到目标路径。
- **预期行为**: 执行移动或复制操作，支持批量操作。
- **输入参数**:
  - `ids`: 源文件/文件夹 ID 数组（字符串数组）
  - `targetPath`: 目标路径（字符串）
  - `operation`: 操作类型（枚举：move, copy）
  - `deviceId`: 外接设备 ID（字符串，可选，指定时表示源或目标涉及外接设备）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 删除文件

- **功能描述**: 删除指定的文件或文件夹（移入回收站）。
- **预期行为**: 将文件移入回收站，支持批量删除。
- **输入参数**:
  - `ids`: 文件/文件夹 ID 数组（字符串数组）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 分享文件/文件夹

- **功能描述**: 创建文件或文件夹的分享链接。
- **预期行为**: 生成可分享的链接，支持设置有效期和访问密码。
- **输入参数**:
  - `id`: 文件/文件夹 ID（字符串）
  - `expireDays`: 有效期天数（数值，可选）
  - `password`: 访问密码（字符串，可选）
  - `permission`: 权限（枚举：read, readwrite）
- **输出数据**:
  - `shareId`: 分享 ID（字符串）
  - `shareUrl`: 分享链接（字符串）
  - `expireAt`: 过期时间（字符串）

### 取消分享

- **功能描述**: 取消已创建的分享。
- **预期行为**: 使分享链接失效。
- **输入参数**:
  - `shareId`: 分享 ID（字符串）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 获取分享详情

- **功能描述**: 获取指定分享的详细信息。
- **预期行为**: 返回分享的权限、有效期、访问统计等信息。
- **输入参数**:
  - `shareId`: 分享 ID（字符串）
- **输出数据**:
  - `id`: 分享 ID（字符串）
  - `resourceName`: 被分享资源名称（字符串）
  - `permission`: 权限（字符串）
  - `expireAt`: 过期时间（字符串）
  - `accessCount`: 访问次数（数值）

### 设置分享权限

- **功能描述**: 修改已创建分享的权限设置。
- **预期行为**: 更新分享的访问权限、有效期或密码。
- **输入参数**:
  - `shareId`: 分享 ID（字符串）
  - `permission`: 新权限（枚举：read, readwrite）
  - `expireDays`: 新有效期（数值，可选）
  - `password`: 新密码（字符串，可选）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 获取我的分享列表

- **功能描述**: 获取当前用户创建的所有分享列表。
- **预期行为**: 返回用户创建的分享记录，包含分享链接信息和统计。
- **输入参数**:
  - `page`: 页码（数值）
  - `pageSize`: 每页条数（数值）
- **输出数据**:
  - `items`: 分享列表，每项包含分享详情结构
  - `total`: 总条数（数值）

### 获取他人分享列表

- **功能描述**: 获取其他用户分享给当前用户的文件列表。
- **预期行为**: 返回接收到的分享记录。
- **输入参数**:
  - `page`: 页码（数值）
  - `pageSize`: 每页条数（数值）
- **输出数据**:
  - `items`: 分享列表，每项包含：
    - `shareId`: 分享 ID（字符串）
    - `resourceName`: 资源名称（字符串）
    - `sharer`: 分享者名称（字符串）
    - `shareTime`: 分享时间（字符串）
  - `total`: 总条数（数值）

### 获取他人给当前用户的分享列表

- **功能描述**: 获取其他用户分享给当前用户的分享列表，按分享者分组，用于"他人分享"页面左侧目录树展示。
- **预期行为**: 返回按分享者分组的分享记录，包含分享资源的层级结构。
- **输入参数**: 无。
- **输出数据**:
  - `groups`: 分享者分组列表，每项包含：
    - `sharer`: 分享者名称（字符串）
    - `sharerId`: 分享者用户 ID（字符串）
    - `shares`: 分享资源列表，每项包含：
      - `shareId`: 分享 ID（字符串）
      - `resourceName`: 资源名称（字符串）
      - `resourceId`: 资源 ID（字符串）
      - `type`: 资源类型（枚举：file, folder）
      - `permission`: 权限（枚举：read, readwrite）
      - `expireAt`: 过期时间（字符串，可选）

### 保存他人分享到我的文件

- **功能描述**: 将他人分享的文件保存到自己的存储空间中。
- **预期行为**: 复制分享的资源到用户指定的本地路径。
- **输入参数**:
  - `shareId`: 分享 ID（字符串）
  - `targetPath`: 保存目标路径（字符串）
- **输出数据**:
  - `success`: 操作是否成功（布尔值）

### 获取公共文件夹列表

- **功能描述**: 获取系统中预定义的公共文件夹分类列表及其层级结构，用于"公共文件"页面左侧目录树展示。
- **预期行为**: 返回所有公共文件夹分类及子目录结构，包含所属存储池标识。
- **输入参数**: 无。
- **输出数据**:
  - `folders`: 公共文件夹列表，每项包含：
    - `id`: 文件夹 ID（字符串）
    - `name`: 文件夹名称（字符串）
    - `poolId`: 所属存储池 ID（字符串）
    - `poolName`: 所属存储池名称（字符串）
    - `children`: 子文件夹数组（可选）

### 创建公共文件夹

- **功能描述**: 创建新的公共文件夹，并绑定到指定存储池。
- **预期行为**: 在系统中创建公共文件夹分类，管理员预定义团队公共资源入口。
- **输入参数**:
  - `name`: 文件夹名称（字符串）
  - `poolId`: 所属存储池 ID（字符串）
- **输出数据**:
  - `id`: 新文件夹 ID（字符串）
  - `name`: 文件夹名称（字符串）
  - `poolId`: 所属存储池 ID（字符串）
  - `createdTime`: 创建时间（字符串）

### 预览文件

- **功能描述**: 预览支持的文件类型（图片、文档、音视频等）。
- **预期行为**: 返回文件的预览数据或预览 URL。
- **输入参数**:
  - `id`: 文件 ID（字符串）

- **输入参数**:

  - `email`: 邮箱（字符串）

  - `role`: 角色（枚举：admin, user）
- **输出数据**:
  - `id`: 新用户 ID（字符串）
  - `username`: 用户名（字符串）

### 获取存储空间列表（系统设置）

    - `poolName`: 所属存储池名称（字符串）
    - `total`: 总容量（字符串）
    - `used`: 已用容量（字符串）
    - `status`: 状态（枚举：normal, warning, error）

### 获取服务列表
