# PRD: 外接设备文件管理

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/external-device/:deviceId` |
| 菜单标题 | 外接设备文件管理（隐藏菜单） |
| 菜单图标 | `lucide:folder-open` |

## 需求概述

外接设备文件管理页展示外接设备的文件目录和本地存储目录，支持在设备与本地之间传输文件。

## 功能行为

### 1. 设备信息头部

显示当前设备的基本信息：
- 设备名称
- 容量与已用空间
- 文件系统类型
- 挂载路径

### 2. 文件操作

两侧均支持：
- 目录树浏览
- 文件列表/网格视图
- 搜索
- 重命名

### 3. 文件传输

- 选中一侧文件后，点击传输按钮
- 弹出传输确认弹窗
- 传输方向：设备 → 本地 或 本地 → 设备

### 4. 路径导航

- 两侧各自独立维护路径
- 支持点击跳转上级目录

## 预期交互

- 选中设备文件，点击"传输到本地"，自动选中目标目录
- 传输完成后刷新两侧文件列表
- 支持单个文件传输

## 数据需求（Mock）

- 外接设备基本信息数据
- 外接设备文件目录树数据
- 外接设备文件列表数据
- 本地存储文件目录树数据
- 本地存储文件列表数据

## 关联页面

- `/file/external-device` — 外接设备列表页（返回入口）
