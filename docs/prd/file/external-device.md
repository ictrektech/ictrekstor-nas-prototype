# PRD: 外接设备

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/external-device` |
| 菜单标题 | 外接设备 |
| 菜单图标 | `lucide:usb` |
| 页面权限 | 已登录用户 |
| 页面类型 | 列表页（主页面） |

## 需求概述

**总体功能**：外接设备页面用于查看和管理通过 USB 或 SD 卡槽连接到 NAS 的外部存储设备，包括 U 盘、移动硬盘、SSD、SD 卡等。以卡片网格形式展示设备列表。

**目标用户**：需要使用外部存储设备与 NAS 交换数据的用户。

**核心价值**：提供外接设备的统一查看入口，方便用户快速了解设备状态和进行文件交换。

---

## 功能规格

### 功能模块: 设备列表展示

#### 功能: 设备卡片网格
- **功能描述**：以卡片网格形式展示所有外接存储设备的基本信息。
- **预期行为**：
  - 每张卡片展示设备类型图标、设备名称、容量与已用空间、使用进度条
  - 显示连接状态（已连接/未连接）、挂载路径、文件系统类型
  - 设备列表为空时显示空状态提示
  - 已连接设备卡片显示可操作样式，未连接设备显示灰色状态
- **约束条件**：
  - 设备信息由系统自动检测，不可手动添加
  - 未连接设备仅显示历史信息，不可操作
- **可用场景**：查看当前连接到 NAS 的所有外部存储设备。
- **涉及接口**：
  - `GET /api/devices/external` — 获取外接设备列表

---

### 功能模块: 设备操作

#### 功能: 打开设备文件
- **功能描述**：点击进入已连接外接设备的文件管理页面。
- **预期行为**：
  - 点击已连接设备卡片，跳转至设备文件管理页
  - 未连接设备卡片不可点击
- **约束条件**：
  - 仅已连接并正确挂载的设备可打开
  - 文件系统不支持的设备可能无法正常浏览
- **可用场景**：访问外接设备中的文件，进行设备与本地 NAS 之间的文件传输。
- **涉及接口**：无（前端路由跳转）

#### 功能: 安全弹出设备
- **功能描述**：安全卸载已连接的外接设备，防止数据损坏。
- **预期行为**：
  - 点击已连接设备卡片上的"安全弹出"按钮
  - 弹出确认提示
  - 确认后系统执行卸载操作
  - 卸载成功后设备状态变为"未连接"
- **约束条件**：
  - 仅已连接设备可弹出
  - 设备正在被访问时（有文件传输或程序占用）可能无法弹出
  - 弹出失败时提示具体原因
- **可用场景**：安全移除 U 盘或移动硬盘。
- **涉及接口**：
  - `POST /api/devices/external/:id/eject` — 安全弹出设备

---

### 功能模块: 设备状态管理

#### 功能: 设备状态显示
- **功能描述**：实时显示外接设备的连接状态。
- **预期行为**：
  - `connected`：已连接并挂载，卡片显示正常样式，可操作
  - `disconnected`：未连接，卡片显示灰色，禁用操作按钮
- **约束条件**：
  - 状态由系统硬件检测驱动，前端定时刷新获取最新状态
- **可用场景**：判断设备是否可用。
- **涉及接口**：
  - `GET /api/devices/external` — 获取设备列表（含状态）

---

## 预期交互

### 场景: 查看外接设备
1. 用户进入"外接设备"页面
2. 页面加载显示所有已知外接设备卡片
3. 已连接的 U 盘显示蓝色正常状态，显示容量 64GB / 已用 32GB
4. 未连接的历史设备显示灰色，提示"未连接"

### 场景: 访问外接设备文件
1. 用户点击已连接的移动硬盘卡片
2. 页面跳转至外接设备文件管理页
3. 用户浏览设备中的文件，选择传输到本地 NAS

### 场景: 安全弹出设备
1. 用户完成文件传输后，回到外接设备列表
2. 点击 U 盘卡片上的"安全弹出"按钮
3. 确认弹窗后设备卸载成功，状态变为未连接
