# PRD: 他人分享

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/shared-from-others` |
| 菜单标题 | 他人分享 |
| 菜单图标 | `lucide:share-2` |
| 页面权限 | 已登录用户 |
| 页面类型 | 文件管理页（主页面） |

## 需求概述

**总体功能**："他人分享"页面展示其他用户共享给当前用户的文件和文件夹，按分享者分组展示，支持浏览、下载和保存到个人空间。

**目标用户**：NAS 系统中接收到他人共享的普通用户。

**核心价值**：集中管理所有来自他人的共享内容，便于查找和使用协作资料。

---

## 功能规格

### 功能模块: 分享目录树

#### 功能: 按分享者分组的目录树
- **功能描述**：以树形结构展示他人共享给当前用户的文件目录，按分享者分组。
- **预期行为**：
  - 一级节点为分享者，显示"来自 xxx"
  - 二级节点为该用户分享的具体文件夹
  - 点击文件夹加载对应文件列表
- **约束条件**：
  - 仅显示当前用户有权限访问的共享内容
  - 共享过期或被撤销后自动从列表移除
- **可用场景**：查看同事或朋友共享给自己的资料。
- **涉及接口**：
  - 获取他人共享给当前用户的目录树

---

### 功能模块: 文件操作

#### 功能: 浏览他人分享的文件
- **功能描述**：在右侧文件面板展示当前选中分享目录的文件列表。
- **预期行为**：
  - 支持列表视图和网格视图切换
  - 单击选中文件，双击预览/下载
  - 列表模式显示文件名、类型、大小、修改时间
- **约束条件**：
  - 文件权限由共享者设置（只读或读写）
  - 当前阶段暂不支持修改他人分享的文件内容（只读模式下）
- **可用场景**：浏览和使用他人共享的资料。
- **涉及接口**：
  - 获取分享目录下的文件列表

#### 功能: 下载文件
- **功能描述**：将他人分享的文件下载到本地。
- **预期行为**：
  - 选中文件后点击"下载"按钮
  - 浏览器触发文件下载
- **约束条件**：
  - 需具备该共享的读取权限
- **可用场景**：将共享资料保存到本地使用。
- **涉及接口**：
  - 下载文件

#### 功能: 保存到我的文件
- **功能描述**：将他人分享的文件或文件夹复制到"我的文件"个人空间。
- **预期行为**：
  - 选中文件后点击"保存到我的文件"按钮
  - 弹出目标目录选择弹窗（从"我的文件"目录树中选择）
  - 确认后将文件复制到目标位置
  - 显示成功提示
- **约束条件**：
  - 目标目录需具备写入权限
  - 同名文件需提示覆盖或重命名
  - 仅复制，不影响原共享文件
- **可用场景**：将重要共享资料备份到个人空间长期保存。
- **涉及接口**：
  - 复制文件到目标目录

#### 功能: 搜索文件
- **功能描述**：在当前分享目录下按文件名搜索。
- **预期行为**：
  - 搜索框输入关键词，回车触发搜索
  - 搜索结果在文件面板展示
- **约束条件**：
  - 仅搜索当前选中的分享目录
- **可用场景**：在大量共享文件中快速定位目标资料。
- **涉及接口**：
  - 搜索分享目录中的文件

## 预期交互

### 场景: 查看同事共享的资料
1. 用户进入"他人分享"页面
2. 左侧目录树加载，显示"来自 张三"、"来自 李四"等分享者节点
3. 用户展开"来自 张三"，点击"项目文档"文件夹
4. 右侧文件面板加载该分享目录的文件列表
5. 用户预览或下载需要的文件

### 场景: 保存共享资料到个人空间
1. 用户在他人分享中选中多个重要文件
2. 点击"保存到我的文件"
3. 弹窗中选择"我的文件 > 工作备份"作为目标目录
4. 确认后文件被复制到个人空间，显示成功提示

## 数据需求（Mock）

- **按分享者分组的目录树数据**：分享者信息、共享文件夹层级结构
- **各分享目录下的文件列表数据**：文件名、类型、大小、修改时间、共享权限
- **用户个人目录树数据**：用于"保存到我的文件"时选择目标目录
