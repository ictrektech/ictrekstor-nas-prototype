## 需求概述

"他人分享"页面展示其他用户共享给当前用户的文件和文件夹。采用文件管理器布局，左侧按分享者分组展示目录树，右侧展示具体文件内容。

## 功能行为

### 1. 页面布局

- **左侧**：目录树，按分享者分组（如"来自 张三"、"来自 李四"）
- **右侧**：文件管理面板，展示选中分享目录下的文件
- **顶部**：面包屑导航 + 搜索框 + 视图切换

### 2. 目录树结构

- 一级节点为分享者（`shared-user` 类型），显示"来自 xxx"
- 二级节点为该用户分享的具体文件夹
- 点击文件夹加载对应文件列表

### 3. 文件操作

- 支持浏览、下载、搜索他人分享的文件
- 支持复制/保存到"我的文件"
- 暂不支持修改他人分享的文件内容

### 4. 视图模式

- 支持列表/网格两种视图切换
- 列表模式显示文件名、类型、大小、修改时间

## 子组件

无页面级专用子组件，复用全局 `FileTreePanel`、`FileManagerPanel`。

## 预期交互

- 选择左侧不同分享者/文件夹，右侧加载对应文件
- 面包屑导航显示当前路径
- 搜索框支持按文件名筛选
