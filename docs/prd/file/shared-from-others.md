# PRD: 他人分享

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/shared-from-others` |
| 菜单标题 | 他人分享 |
| 菜单图标 | `lucide:share-2` |

## 需求概述

"他人分享"页面展示其他用户共享给当前用户的文件和文件夹。按分享者分组展示目录树，展示具体文件内容。

## 功能行为

### 1. 目录树结构

- 一级节点为分享者，显示"来自 xxx"
- 二级节点为该用户分享的具体文件夹
- 点击文件夹加载对应文件列表

### 2. 文件操作

- 支持浏览、下载、搜索他人分享的文件
- 支持复制/保存到"我的文件"
- 暂不支持修改他人分享的文件内容

### 3. 视图模式

- 支持列表/网格两种视图切换
- 列表模式显示文件名、类型、大小、修改时间

## 预期交互

- 选择不同分享者/文件夹，加载对应文件
- 路径导航显示当前路径
- 搜索框支持按文件名筛选

## 数据需求（Mock）

- 按分享者分组的目录树数据
- 各分享目录下的文件列表数据

## 关联页面

- 复用全局文件管理组件
- "我的分享"为反向功能（当前用户分享给他人）
