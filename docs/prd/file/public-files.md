# PRD: 公共文件

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/public-files` |
| 菜单标题 | 公共文件 |
| 菜单图标 | `lucide:folder-heart` |

## 需求概述

公共文件页面用于管理团队级别的公共文件资源，所有用户均可访问。文件树以"公共文件夹"为根节点，每个公共文件夹绑定到特定的存储池。

## 功能行为

### 1. 目录树

- 根节点为预定义的公共文件夹分类（如"文档"、"设计资源"、"财务资料"、"培训资料"）
- 每个公共文件夹节点标识所属存储池

### 2. 创建公共文件夹

- 点击"创建公共文件夹"按钮，弹出表单弹窗
- 表单字段：文件夹名称（必填）、所属存储池（下拉选择，必填）
- 存储池选项从系统存储池列表中获取

### 3. 文件操作

- 支持浏览、下载、搜索公共文件
- 当前阶段暂不支持在公共文件夹内创建子文件夹/上传文件（STUB）

### 4. 存储池显示

- 页面头部显示当前选中文件夹的所属存储池
- 存储池映射关系由系统维护

## 预期交互

- 选择不同公共文件夹，显示对应文件列表
- 路径导航显示当前路径
- 创建公共文件夹后自动刷新目录树

## 数据需求（Mock）

- 公共文件夹目录树数据
- 各公共文件夹下的文件列表数据
- 系统存储池列表数据（用于创建公共文件夹时选择）
