## 需求概述

公共文件页面用于管理团队级别的公共文件资源，所有用户均可访问。采用与"我的文件"相同的文件管理器布局，但文件树以"公共文件夹"为根节点，每个公共文件夹绑定到特定的存储池。

## 功能行为

### 1. 页面布局

- 同"我的文件"：左侧目录树 + 右侧文件管理面板
- 顶部显示"所属存储池"标识

### 2. 目录树

- 根节点为预定义的公共文件夹分类（如"文档"、"设计资源"、"财务资料"、"培训资料"）
- 每个公共文件夹节点包含 `storagePool` 属性，标识所属存储池

### 3. 创建公共文件夹

- 点击"创建公共文件夹"按钮，弹出表单弹窗
- 表单字段：文件夹名称（必填）、所属存储池（下拉选择，必填）
- 存储池选项从预定义列表中获取

### 4. 文件操作

- 支持浏览、下载、搜索公共文件
- 当前阶段暂不支持在公共文件夹内创建子文件夹/上传文件（STUB）

### 5. 存储池显示

- 页面头部右侧显示当前选中文件夹的所属存储池
- 存储池映射关系硬编码在 `poolLabelMap` 中

## 子组件

无页面级专用子组件，复用全局 `FileTreePanel`、`FileManagerPanel`。

## 预期交互

- 选择左侧不同公共文件夹，右侧显示对应文件列表
- 面包屑导航显示当前路径
- 创建公共文件夹后自动刷新目录树
