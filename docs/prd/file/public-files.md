# PRD: 公共文件

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/public-files` |
| 菜单标题 | 公共文件 |
| 菜单图标 | `lucide:folder-heart` |
| 页面权限 | 已登录用户 |
| 页面类型 | 文件管理页（主页面） |

## 需求概述

**总体功能**：公共文件页面用于管理团队级别的公共文件资源，所有用户均可访问。提供公共文件夹的浏览、创建、以及文件下载等功能。

**目标用户**：NAS 系统的所有用户，需要访问团队共享的公共资料。

**核心价值**：为团队提供统一的公共资源访问入口，便于知识共享和资料协作。

---

## 功能规格

### 功能模块: 公共文件夹目录树

#### 功能: 公共文件夹目录树
- **功能描述**：以树形结构展示预定义的公共文件夹分类及其子目录。
- **预期行为**：
  - 根节点为预定义的公共文件夹分类（如"文档"、"设计资源"、"财务资料"、"培训资料"）
  - 每个公共文件夹节点标识所属存储池
  - 支持展开/收起子目录
  - 点击节点加载对应文件列表
- **约束条件**：
  - 公共文件夹由管理员预定义，普通用户不可随意创建顶级分类
  - 各公共文件夹绑定到特定存储池
- **可用场景**：浏览团队公共资源目录，定位需要的公共资料。
- **涉及接口**：
  - `GET /api/file/public/tree` — 获取公共文件夹目录树

---

### 功能模块: 文件操作

#### 功能: 文件浏览
- **功能描述**：在右侧文件面板展示当前公共文件夹的文件列表。
- **预期行为**：
  - 支持列表视图和网格视图
  - 单击选中文件，双击预览/下载
  - 显示文件名、类型、大小、修改时间
- **约束条件**：
  - 仅支持浏览和下载，当前阶段暂不支持在公共文件夹内创建子文件夹/上传文件
  - 文件操作权限受所属存储池的访问控制策略限制
- **可用场景**：查找和浏览团队共享的文档、图片、视频等资料。
- **涉及接口**：
  - `GET /api/file/public/list` — 获取公共文件夹文件列表

#### 功能: 文件下载
- **功能描述**：将公共文件下载到本地。
- **预期行为**：
  - 选中文件后点击"下载"按钮
  - 浏览器触发文件下载
- **约束条件**：
  - 需具备该公共文件夹的读取权限
- **可用场景**：将公共资料取回本地参考使用。
- **涉及接口**：
  - `GET /api/file/download` — 下载文件

#### 功能: 文件搜索
- **功能描述**：在当前公共文件夹下按文件名搜索。
- **预期行为**：
  - 搜索框输入关键词，回车触发搜索
  - 搜索结果在文件面板展示
- **约束条件**：
  - 仅搜索当前选中的公共文件夹及其子目录
- **可用场景**：在大量公共文件中快速定位目标资料。
- **涉及接口**：
  - `GET /api/file/public/search` — 搜索公共文件

---

### 功能模块: 创建公共文件夹

#### 功能: 新建公共文件夹
- **功能描述**：创建新的公共文件夹，并绑定到指定存储池。
- **预期行为**：
  - 点击"创建公共文件夹"按钮，弹出表单弹窗
  - 表单字段：文件夹名称（必填）、所属存储池（下拉选择，必填）
  - 存储池选项从系统存储池列表中获取
  - 确认后创建公共文件夹，刷新目录树
- **约束条件**：
  - 仅具备管理员权限或特定角色的用户可创建公共文件夹
  - 文件夹名称不能为空，不能包含特殊字符
  - 所属存储池必须存在且可用
- **可用场景**：管理员为团队新增公共资料分类。
- **涉及接口**：
  - `POST /api/file/public/mkdir` — 创建公共文件夹
  - `GET /api/storage/pools` — 获取可用存储池列表

---

### 功能模块: 存储池信息展示

#### 功能: 当前存储池信息
- **功能描述**：页面头部显示当前选中公共文件夹的所属存储池信息。
- **预期行为**：
  - 选择不同公共文件夹时，头部显示对应的存储池名称
  - 存储池映射关系由系统维护
- **约束条件**：仅展示信息，不可在此修改映射关系。
- **可用场景**：了解公共文件夹的资源归属。
- **涉及接口**：无（随目录树数据返回）

---

## 预期交互

### 场景: 浏览公共资料
1. 用户进入"公共文件"页面
2. 左侧目录树加载公共文件夹分类
3. 用户点击"培训资料"文件夹
4. 右侧文件面板加载该文件夹下的文件列表
5. 用户点击文件进行预览或下载

### 场景: 创建新的公共文件夹
1. 管理员点击"创建公共文件夹"按钮
2. 弹窗中输入文件夹名称"项目文档"
3. 从下拉列表选择所属存储池"存储空间1"
4. 确认创建，目录树自动更新并展开新文件夹

## 数据需求（Mock）

- **公共文件夹目录树数据**：预定义分类及层级结构，含所属存储池标识
- **公共文件夹文件列表数据**：文件名、类型、大小、修改时间
- **系统存储池列表数据**：存储池名称、ID、健康状态（用于创建时选择）
