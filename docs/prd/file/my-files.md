# PRD: 我的文件

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/my-files` |
| 菜单标题 | 我的文件 |
| 菜单图标 | `lucide:folder-open` |

## 需求概述

"我的文件"是用户个人文件空间的管理界面，支持文件的上传、下载、新建、重命名、复制、移动、删除、共享等完整操作。

## 功能行为

### 1. 目录树

- 根节点为存储空间（如"存储空间1"、"存储空间2"）
- 支持展开/收起文件夹
- 点击文件夹加载对应文件列表
- 树节点支持自定义图标（空间图标、文件夹图标）

### 2. 文件操作

| 操作 | 行为 |
|------|------|
| 单击文件 | 选中/取消选中 |
| 双击文件 | 文件夹则进入目录，文件则预览/下载 |
| 右键菜单 | 重命名、复制、移动、删除、下载、共享、属性 |
| 新建文件夹 | 弹窗输入名称，在当前目录创建 |
| 上传文件 | 通过文件选择器上传，支持拖拽 |
| 搜索 | 按文件名模糊搜索当前目录 |

### 3. 批量操作

- 支持多选文件（Ctrl/Shift + 点击）
- 批量删除、批量复制、批量移动

### 4. 共享设置

- 选中文件/文件夹后点击共享，弹出共享配置弹窗
- 支持设置共享用户、权限（读/写）、有效期
- 支持生成共享链接

### 5. 复制/移动

- 选择目标后弹出"选择目标目录"弹窗
- 显示目录树供用户选择目标位置

## 预期交互

- 文件列表支持列表/网格两种视图切换
- 支持拖拽上传文件到当前目录
- 复制/移动操作后显示成功提示并刷新文件列表
- 路径导航支持点击跳转上级目录

## 数据需求（Mock）

- 目录树数据：存储空间及文件夹层级结构
- 文件列表数据：文件名、类型、大小、修改时间等
- 存储空间列表数据

## 关联页面

- 共享设置复用全局共享配置弹窗组件
- 重命名复用全局重命名弹窗组件
