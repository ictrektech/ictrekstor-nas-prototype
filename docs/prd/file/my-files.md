## 需求概述

"我的文件"是用户个人文件空间的管理界面，采用经典的文件管理器布局（左侧目录树 + 右侧文件列表/网格），支持文件的上传、下载、新建、重命名、复制、移动、删除、共享等完整操作。

## 功能行为

### 1. 页面布局

- **左侧**：目录树（`FileTreePanel`），展示用户存储空间及文件夹层级
- **右侧**：文件管理面板（`FileManagerPanel`），展示当前选中目录的文件列表
- **顶部**：面包屑导航 + 搜索框 + 视图切换（列表/网格）+ 批量操作工具栏

### 2. 目录树

- 根节点为存储空间（如"存储空间1"、"存储空间2"）
- 支持展开/收起文件夹
- 点击文件夹加载对应文件列表
- 树节点支持自定义图标（空间图标、文件夹图标）

### 3. 文件操作

| 操作 | 行为 |
|------|------|
| 单击文件 | 选中/取消选中 |
| 双击文件 | 文件夹则进入目录，文件则预览/下载 |
| 右键菜单 | 重命名、复制、移动、删除、下载、共享、属性 |
| 新建文件夹 | 弹窗输入名称，在当前目录创建 |
| 上传文件 | 通过文件选择器上传，支持拖拽 |
| 搜索 | 按文件名模糊搜索当前目录 |

### 4. 批量操作

- 支持多选文件（Ctrl/Shift + 点击）
- 批量删除、批量复制、批量移动

### 5. 共享设置

- 选中文件/文件夹后点击共享，弹出共享配置弹窗
- 支持设置共享用户、权限（读/写）、有效期
- 支持生成共享链接

### 6. 复制/移动

- 选择目标后弹出"选择目标目录"弹窗
- 显示目录树供用户选择目标位置
