# PRD: 我的文件

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/my-files` |
| 菜单标题 | 我的文件 |
| 菜单图标 | `lucide:folder-open` |
| 页面权限 | 已登录用户 |
| 页面类型 | 文件管理页（主页面） |

## 需求概述

**总体功能**："我的文件"是用户个人私有文件空间的管理界面，提供目录树浏览、文件上传下载、文件操作（新建/重命名/复制/移动/删除）、共享配置等完整的文件管理功能。

**目标用户**：NAS 系统的普通用户，需要管理个人文件资料。

**核心价值**：为用户提供安全、便捷的个人文件存储与管理入口，是个人工作的核心文件空间。

---

## 功能规格

### 功能模块: 目录树导航

#### 功能: 存储空间目录树
- **功能描述**：以树形结构展示用户的存储空间及文件夹层级关系。
- **预期行为**：
  - 根节点为各存储空间（如"存储空间1"）
  - 点击文件夹节点展开/收起下级目录
  - 点击节点加载对应目录的文件列表到右侧文件面板
  - 树节点使用不同图标区分存储空间和文件夹
- **约束条件**：
  - 仅展示当前用户有权限访问的存储空间
  - 深层嵌套目录（>10层）可能延迟加载
- **可用场景**：浏览个人文件目录结构，快速定位到目标文件夹。
- **涉及接口**：
  - `GET /api/file/tree` — 获取用户目录树结构

#### 功能: 路径面包屑导航
- **功能描述**：在文件面板顶部显示当前目录的路径导航条。
- **预期行为**：
  - 显示从存储空间根目录到当前目录的完整路径
  - 点击路径中的任意节点可跳转到对应目录
  - 路径过长时中间节点可折叠为省略号
- **约束条件**：面包屑仅反映已加载的目录层级。
- **可用场景**：在深层目录中快速返回上级目录。
- **涉及接口**：无（纯前端状态管理）

---

### 功能模块: 文件操作

#### 功能: 文件浏览与选择
- **功能描述**：在右侧文件面板展示当前目录的文件列表，支持文件选择和预览。
- **预期行为**：
  - 支持列表视图和网格视图两种展示模式
  - 单击文件/文件夹可选中/取消选中（支持 Ctrl/Shift 多选）
  - 双击文件夹进入该目录
  - 双击文件触发预览或下载
  - 右键点击文件弹出上下文操作菜单
- **约束条件**：
  - 空文件夹显示空状态提示
  - 文件列表按名称/大小/修改时间支持排序
- **可用场景**：浏览、查找、选择目标文件。
- **涉及接口**：
  - `GET /api/file/list` — 获取当前目录文件列表

#### 功能: 新建文件夹
- **功能描述**：在当前目录下创建新的文件夹。
- **预期行为**：
  - 点击"新建文件夹"按钮，弹出输入弹窗
  - 输入文件夹名称，点击确认创建
  - 名称重复时提示错误
  - 创建成功后刷新文件列表并自动展开目录树对应节点
- **约束条件**：
  - 文件夹名称不能为空，不能包含特殊字符（`\/:*?"<>|`）
  - 同目录下名称不可重复
  - 当前用户需具备该目录的写入权限
- **可用场景**：组织文件结构，创建新的分类目录。
- **涉及接口**：
  - `POST /api/file/mkdir` — 创建文件夹

#### 功能: 文件重命名
- **功能描述**：修改文件或文件夹的名称。
- **预期行为**：
  - 通过右键菜单或工具栏触发重命名弹窗
  - 输入新名称，确认后完成重命名
  - 重命名成功后刷新文件列表
- **约束条件**：
  - 新名称不能与同目录下其他文件/文件夹重名
  - 名称不能为空，不能包含特殊字符
  - 重命名操作不影响文件内容和路径引用
- **可用场景**：修正文件命名错误或优化文件组织结构。
- **涉及接口**：
  - `PUT /api/file/rename` — 重命名文件或文件夹

#### 功能: 文件复制与移动
- **功能描述**：将选中的文件或文件夹复制或移动到目标目录。
- **预期行为**：
  - 选择文件后点击"复制"或"移动"按钮
  - 弹出"选择目标目录"弹窗，展示目录树供选择
  - 选择目标目录后确认执行
  - 操作完成后显示成功提示，刷新文件列表
  - 支持跨存储空间复制/移动
- **约束条件**：
  - 目标目录不能是源文件本身的子目录（防止循环）
  - 目标目录同名文件需提示覆盖或重命名
  - 移动操作需源目录写入权限和目标目录写入权限
- **可用场景**：整理文件位置、备份文件到其他存储空间。
- **涉及接口**：
  - `POST /api/file/copy` — 复制文件或文件夹
  - `POST /api/file/move` — 移动文件或文件夹

#### 功能: 文件删除
- **功能描述**：将选中的文件或文件夹移入回收站。
- **预期行为**：
  - 选中文件后点击"删除"按钮
  - 弹出二次确认弹窗，提示删除的文件数量
  - 确认后文件移入回收站，显示成功提示
  - 刷新文件列表
- **约束条件**：
  - 删除文件夹会递归删除其下所有内容
  - 需具备源目录的删除权限
  - 移入回收站后可还原，非永久删除
- **可用场景**：清理不再需要的文件或文件夹。
- **涉及接口**：
  - `DELETE /api/file/delete` — 移入回收站

#### 功能: 文件上传
- **功能描述**：将本地文件上传到当前目录。
- **预期行为**：
  - 点击"上传"按钮打开文件选择器
  - 支持拖拽文件到文件面板区域上传
  - 显示上传进度条
  - 上传完成后刷新文件列表
  - 支持多文件同时上传
- **约束条件**：
  - 单文件大小上限由系统配置决定（默认 4GB）
  - 当前目录需具备写入权限
  - 同名文件提示覆盖或重命名
- **可用场景**：将本地资料备份到 NAS 个人空间。
- **涉及接口**：
  - `POST /api/file/upload` — 上传文件（支持分片）

#### 功能: 文件下载
- **功能描述**：将 NAS 上的文件下载到本地。
- **预期行为**：
  - 选中文件后点击"下载"按钮
  - 浏览器触发文件下载
  - 支持批量下载（打包为 ZIP）
- **约束条件**：
  - 需具备文件的读取权限
  - 大文件下载可能需要较长时间
- **可用场景**：将 NAS 文件取回本地使用。
- **涉及接口**：
  - `GET /api/file/download` — 下载单个文件
  - `POST /api/file/download/batch` — 批量下载（返回 ZIP）

#### 功能: 文件搜索
- **功能描述**：按文件名在当前目录下搜索文件。
- **预期行为**：
  - 在搜索框输入关键词，实时或回车触发搜索
  - 搜索结果在文件面板中展示
  - 支持清空搜索回到完整列表
- **约束条件**：
  - 仅搜索当前目录及其子目录（或仅当前目录，取决于配置）
  - 支持模糊匹配
- **可用场景**：在大量文件中快速定位目标文件。
- **涉及接口**：
  - `GET /api/file/search` — 按名称搜索文件

---

### 功能模块: 共享设置

#### 功能: 创建共享
- **功能描述**：将选中的文件或文件夹共享给其他用户或生成公开链接。
- **预期行为**：
  - 选中文件后点击"共享"，弹出共享配置弹窗
  - 配置共享用户列表及权限（只读/读写）
  - 支持生成公开分享链接
  - 可设置共享有效期
  - 配置完成后显示共享成功提示
- **约束条件**：
  - 仅文件所有者或具备管理权限的用户可创建共享
  - 共享用户必须是系统中存在的用户
  - 有效期到期后共享自动失效
- **可用场景**：将工作文档共享给同事协作。
- **涉及接口**：
  - `POST /api/share/create` — 创建共享

---

### 功能模块: 批量操作

#### 功能: 批量文件操作
- **功能描述**：同时对多个选中的文件执行相同操作。
- **预期行为**：
  - 使用 Ctrl/Shift + 点击选中多个文件
  - 选中后工具栏显示批量操作按钮（删除、复制、移动、下载）
  - 批量删除时弹出确认弹窗，列出待删除文件数量
- **约束条件**：
  - 批量操作涉及的不同文件可能位于不同权限目录
  - 任一文件权限不足则整个操作失败并提示
- **可用场景**：批量整理、清理或备份多个文件。
- **涉及接口**：同对应单文件操作接口，批量传参。

## 预期交互

### 场景: 日常文件浏览
1. 用户进入"我的文件"页面
2. 页面左侧加载目录树，默认展开第一个存储空间
3. 右侧文件面板加载根目录文件列表
4. 用户点击文件夹进入子目录，面包屑同步更新
5. 用户在搜索框输入关键词，实时过滤当前目录文件

### 场景: 上传新文件
1. 用户导航到目标目录
2. 点击"上传"按钮或拖拽文件到面板
3. 文件选择器打开，用户选择本地文件
4. 上传进度条显示，完成后文件出现在列表中

### 场景: 共享文件给同事
1. 用户选中要共享的文件
2. 点击"共享"按钮，打开共享配置弹窗
3. 搜索并添加共享用户，设置权限为"只读"
4. 点击确认，系统生成共享并显示成功提示

### 场景: 整理文件结构
1. 用户选中多个文件
2. 点击"移动到"，弹出目标目录选择弹窗
3. 在目录树中选择目标文件夹
4. 确认后文件从当前目录消失，出现在目标目录

## 数据需求（Mock）

- **目录树数据**：存储空间列表及文件夹层级结构
  ```json
  {
    "id": "space-1",
    "name": "存储空间1",
    "type": "volume",
    "children": [
      {
        "id": "folder-1",
        "name": "工作文档",
        "type": "folder",
        "children": []
      }
    ]
  }
  ```
- **文件列表数据**：文件名、类型、大小、修改时间、权限信息
- **存储空间列表数据**：名称、总容量、已用容量
- **用户列表数据**：用户名、昵称（用于共享配置时选择用户）
