# PRD: 设备全部文件

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/all-files` |
| 菜单标题 | 设备全部文件 |
| 菜单图标 | `lucide:hard-drive` |

## 需求概述

"设备全部文件"页面提供管理员视角的全局文件浏览功能，可查看 NAS 设备上所有存储空间中的文件和文件夹。支持管理员级别的文件操作。

## 功能行为

### 1. 目录树结构

- 根节点为存储空间（如"存储空间1"、"存储空间2"）
- 每个存储空间下包含标准文件夹（文档、图片、视频、音乐、软件等）
- 支持多级文件夹嵌套

### 2. 文件操作

支持管理员级别的完整文件操作：
- 浏览所有存储空间的文件
- 新建文件夹
- 重命名文件/文件夹
- 搜索文件
- 下载文件

### 3. 新建文件夹

- 点击"新建文件夹"按钮
- 弹出新建文件夹弹窗，输入文件夹名称
- 在当前选中目录下创建新文件夹

### 4. 重命名

- 支持文件和文件夹重命名
- 弹出重命名弹窗，输入新名称

## 预期交互

- 选择不同存储空间/文件夹，加载对应文件
- 导航支持跨存储空间跳转
- 搜索框支持按文件名筛选当前目录

## 数据需求（Mock）

- 全局存储空间目录树数据
- 各目录下的文件列表数据

## 关联页面

- 复用全局文件管理组件
- 新建文件夹复用全局新建文件夹弹窗
