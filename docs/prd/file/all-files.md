## 需求概述

"设备全部文件"页面提供管理员视角的全局文件浏览功能，可查看 NAS 设备上所有存储空间中的文件和文件夹。采用文件管理器布局，支持管理员级别的文件操作。

## 功能行为

### 1. 页面布局

- **左侧**：目录树，展示所有存储空间及其子文件夹
- **右侧**：文件管理面板，展示选中目录的文件
- **顶部**：面包屑导航 + 搜索框 + 视图切换

### 2. 目录树结构

- 根节点为存储空间（如"存储空间1"、"存储空间2"）
- 每个存储空间下包含标准文件夹（文档、图片、视频、音乐、软件等）
- 支持多级文件夹嵌套

### 3. 文件操作

支持管理员级别的完整文件操作：
- 浏览所有存储空间的文件
- 新建文件夹
- 重命名文件/文件夹
- 搜索文件
- 下载文件

### 4. 新建文件夹

- 点击"新建文件夹"按钮
- 弹出 `NewFolderModal` 输入文件夹名称
- 在当前选中目录下创建新文件夹

### 5. 重命名

- 支持文件和文件夹重命名
- 弹出重命名弹窗，输入新名称
