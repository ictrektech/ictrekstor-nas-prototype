# PRD: 我的分享

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/file/shared` |
| 菜单标题 | 我的分享 |
| 菜单图标 | `lucide:link` |
| 页面权限 | 已登录用户 |
| 页面类型 | 管理页（主页面） |

## 需求概述

**总体功能**："我的分享"页面用于管理当前用户创建的所有共享目录，提供统计概览、创建新分享、编辑共享设置、管理共享用户和链接等功能。

**目标用户**：需要将文件或文件夹共享给其他用户的 NAS 使用者。

**核心价值**：统一管理所有个人创建的共享，方便追踪共享状态、管理访问权限和控制共享生命周期。

---

## 功能规格

### 功能模块: 统计概览

#### 功能: 共享统计卡片
- **功能描述**：在页面顶部展示当前用户创建的共享关键指标。
- **预期行为**：
  - 显示有效共享数、有效链接数、共享用户数（去重）、共享目录总数
  - 统计数据随共享列表变化实时更新
- **约束条件**：
  - 仅统计当前用户创建的共享
  - 过期的共享不计入"有效"统计
- **可用场景**：快速了解个人共享的整体情况。
- **涉及接口**：
  - 获取当前用户的共享统计数据

---

### 功能模块: 共享目录管理

#### 功能: 共享列表展示
- **功能描述**：以表格形式展示当前用户创建的所有共享目录。
- **预期行为**：
  - 表格列：共享名称、源路径、共享用户列表、链接状态、有效期、操作
  - 支持按名称、路径、用户、链接状态搜索过滤
  - 支持分页展示
  - 点击共享名称可展开或跳转详情
- **约束条件**：
  - 仅展示当前用户创建的共享
  - 表格数据量大时默认分页
- **可用场景**：查看和管理所有个人创建的共享。
- **涉及接口**：
  - 获取当前用户的共享列表

#### 功能: 创建分享
- **功能描述**：创建新的文件或文件夹共享。
- **预期行为**：
  - 点击"创建分享"按钮，弹出分享配置弹窗
  - 第一步：选择源目录（从"我的文件"目录树中选择）
  - 第二步：配置共享用户（搜索并添加用户）及权限（读/写）
  - 第三步：设置共享名称、有效期、是否生成公开链接
  - 确认后创建共享，刷新列表
- **约束条件**：
  - 源目录必须是"我的文件"下的有效路径
  - 共享用户必须是系统中存在的用户
  - 共享名称不能为空
- **可用场景**：将工作目录共享给团队成员协作。
- **涉及接口**：
  - 创建共享
  - 获取"我的文件"目录树（用于选择源目录）
  - 获取用户列表（用于添加共享用户）

#### 功能: 编辑共享用户
- **功能描述**：修改已创建共享的用户列表和权限。
- **预期行为**：
  - 点击共享行的"管理用户"，弹出编辑用户弹窗
  - 可添加新的共享用户
  - 可删除已有共享用户
  - 可修改用户权限（只读/读写）
  - 保存后即时生效
- **约束条件**：
  - 仅共享创建者可管理用户
  - 至少保留一个共享用户（或改为仅链接共享）
- **可用场景**：新增协作成员或调整成员权限。
- **涉及接口**：
  - 更新共享用户列表

#### 功能: 链接管理
- **功能描述**：管理共享的公开链接设置。
- **预期行为**：
  - 点击共享行的"链接管理"，弹出链接管理弹窗
  - 可启用/禁用公开链接
  - 可设置/修改链接密码
  - 可复制链接地址到剪贴板
  - 可重新生成链接（使旧链接失效）
- **约束条件**：
  - 仅共享创建者可管理链接
  - 链接密码为空表示无需密码访问
- **可用场景**：生成链接分享给外部人员，或撤销已发出的链接。
- **涉及接口**：
  - 更新链接配置

#### 功能: 删除共享
- **功能描述**：删除已创建的共享，使其失效。
- **预期行为**：
  - 点击共享行的"删除"按钮
  - 弹出二次确认弹窗
  - 确认后删除共享，共享用户和链接立即失效
  - 刷新列表，不影响源文件
- **约束条件**：
  - 仅共享创建者可删除
  - 删除后不可恢复，需重新创建
- **可用场景**：项目结束或共享不再必要时清理共享。
- **涉及接口**：
  - 删除共享

#### 功能: 复制链接
- **功能描述**：将共享的公开链接地址复制到剪贴板。
- **预期行为**：
  - 点击"复制链接"按钮
  - 链接地址复制到剪贴板
  - 显示复制成功提示
- **约束条件**：
  - 仅当公开链接已启用时可复制
- **可用场景**：快速获取共享链接发送给协作者。
- **涉及接口**：无（纯前端操作）

## 预期交互

### 场景: 创建新分享
1. 用户点击"创建分享"按钮
2. 弹窗第一步：用户在"我的文件"目录树中选择"项目文档"文件夹
3. 弹窗第二步：搜索添加用户"张三"，设置权限为"读写"
4. 弹窗第三步：输入共享名称"项目协作"，设置有效期30天，启用公开链接
5. 确认创建，列表自动刷新显示新共享

### 场景: 管理现有共享
1. 用户在共享列表中找到"项目协作"
2. 点击"管理用户"，添加新成员"李四"
3. 点击"链接管理"，复制公开链接发送给外部合作伙伴
4. 项目结束后，点击"删除"撤销该共享

### 场景: 搜索共享
1. 用户在搜索框输入"项目"
2. 列表实时过滤，仅显示名称或路径包含"项目"的共享

## 数据需求（Mock）

- **共享目录列表数据**：共享ID、名称、源路径、用户列表（含权限）、链接状态、有效期、创建时间
- **用户列表数据**：用户ID、用户名、昵称、头像（用于配置共享用户时搜索选择）
- **"我的文件"目录树数据**：用于创建共享时选择源目录
