# PRD: 智能对话 / AI Chat

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/ai/chat` |
| 页面组件 | `apps/web-antd/src/views/ai/chat/index.vue` |
| 菜单标题 | AI 助手 > 智能对话 |
| 菜单图标 | `lucide:message-circle` |
| 当前状态 | 🚧 施工中（占位页） |

## 需求概述

智能对话页面提供基于 AI 的交互式问答功能，用户可以通过自然语言与 NAS 系统交互，获取设备信息、执行管理操作或获得技术支持。

## 功能行为

### 当前实现

页面目前为**施工占位页**，显示：
- 大号施工图标（`lucide:hard-hat`）
- "施工中" 标题
- "此功能正在开发中，敬请期待" 描述文本

### 预期功能（待实现）

1. **对话界面**：类似 ChatGPT 的聊天窗口，支持多轮对话
2. **历史会话**：左侧会话列表，支持新建/删除/重命名会话
3. **快捷指令**：预设常用命令（如"查看磁盘状态"、"重启 SMB 服务"）
4. **上下文感知**：AI 能够访问当前设备状态数据，提供个性化回答
5. **代码/命令高亮**：AI 返回的命令或配置代码支持语法高亮
6. **文件引用**：支持在对话中引用 NAS 中的文件

## 预期交互

- 用户在输入框输入问题，按 Enter 发送
- AI 流式返回回答内容
- 支持 Markdown 渲染（表格、列表、代码块等）
- 支持复制 AI 回答中的代码片段

## 相关接口

| 接口 | 说明 |
|------|------|
| `POST /api/ai/chat` | 发送对话消息 |
| `GET /api/ai/sessions` | 获取会话列表 |
| `DELETE /api/ai/sessions/:id` | 删除会话 |
