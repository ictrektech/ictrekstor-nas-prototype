# PRD: 首页 / Dashboard

## 页面概要

| 属性 | 值 |
|------|-----|
| 路由 | `/dashboard` |
| 菜单标题 | 首页 |
| 菜单图标 | `lucide:home` |

## 需求概述

首页是用户登录 NAS 管理后台后的默认着陆页，提供设备运行状态、资源使用概况、快捷入口等核心信息的聚合展示。

## 功能行为

1. **设备概览**：展示设备名称、系统版本、运行时长、序列号等核心信息
2. **资源监控**：CPU / 内存 / 磁盘使用率的实时图表
3. **存储概览**：总容量、已用容量、存储池健康状态
4. **网络状态**：各网口连接状态、实时带宽
5. **快捷操作**：常用功能入口（文件管理、存储管理、系统设置等）
6. **系统通知**：最近告警/通知列表

## 预期交互

- 页面加载时自动获取设备摘要数据
- 图表数据每 5 秒自动刷新
- 点击快捷入口跳转到对应功能模块

## 数据需求（Mock）

- 设备摘要信息（设备名称、系统版本、运行时长、序列号等）
- 实时性能指标（CPU/内存/磁盘使用率时间序列数据）
- 存储池状态数据
- 网络接口状态及流量数据
- 系统通知/告警列表

## 关联页面

- 所有一级菜单模块均为首页的快捷入口目标
