## ADDED Requirements

### Requirement: Page header layout unification
All sub-pages under file management and storage management modules SHALL adopt a unified `page-header` layout at the top of the page.

#### Scenario: Standard page header rendered
- **WHEN** a user navigates to any sub-page under file or storage management
- **THEN** the page SHALL display a `page-header` containing:
  - A page icon in a rounded box on the left
  - A page title and description text
  - Statistical overview cards on the right (when applicable)
